# 📊 김순하 포트폴리오 | KSH Portfolio

> 데이터 분석 개발자 김순하의 개인 포트폴리오 웹사이트입니다.

🌐 **Live:** https://ksh1004.github.io/portfolio

---

## 소개

데이터에서 인사이트를 발굴하고, 이를 실질적인 비즈니스 가치로 연결하는 데이터 분석 개발자입니다.

## 구성

| 섹션 | 내용 |
|------|------|
| 자기소개 | 개인 소개 및 핵심 역량 |
| 요약 | 주요 전문성 Overview |
| 학력 | 인하대학교, 북일고등학교 |
| 경력 | 대한민국 육군, 대외 경력 |
| 경험 | 활동, 교육 수료, 대외활동 |
| 자격증 | 보유 자격증 목록 |
| 기술 | 언어, 분석, 시각화, 인프라 |
| 수상 | 수상 이력 |
| 프로젝트 | 주요 프로젝트 포트폴리오 |
| 연락처 | 이메일, GitHub, LinkedIn 등 |

## 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 바로 실행됩니다.

```bash
git clone https://github.com/ksh1004/portfolio.git
cd portfolio
# index.html을 브라우저로 열기
```

## 업데이트

```bash
git add index.html
git commit -m "내용 수정"
git push
```

push 후 약 1~2분 내 사이트에 자동 반영됩니다.
